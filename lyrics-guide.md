## ワンパターンを避けるための変数（毎曲シャッフルする）

以下の各軸について、**毎回意識的に違う選択肢を選ぶ**こと。「いつも通り」に流れない。

### A. 語り手の視点（毎曲変える）

| 視点 | 説明 |
|------|------|
| 一人称「私/僕」 | 主人公が自分の気持ちを語る（王道だが使いすぎに注意） |
| 二人称「あなた」 | 誰かに向けて語りかける。リスナーへの直接訴求 |
| 三人称 | 二人の関係を外から描写する。映画的・物語的 |
| 物・生き物の視点 | 桜、月、猫、風、電車などが語る。意外性と独自世界観 |
| 複数形「私たち」 | 集合的な声。連帯感・壮大さが出る |

### B. Aメロの切り口（毎曲変える）

- **マクロ→ミクロ**：広い情景から始めて主人公にズームイン
- **ミクロ→マクロ**：小さな動作・感覚から始めて世界を広げる
- **特定の瞬間から**：「あの日の朝」「3時15分」など時間をピンポイントで切り取る
- **対話・セリフから**：誰かの言葉をAメロ冒頭に置く
- **視覚以外の感覚から**：音・におい・触感・温度で描写を始める

### C. 感情の描き方（直接か間接か）

**間接法を積極的に使う。** 感情を言葉にせず、別のもので表現する。

- 悲しみ → 「窓の結露が止まらない」
- 胸の高鳴り → 「信号が全部青に見えた」
- 孤独 → 「コンビニの照明だけが友達だった」
- 幸福感 → 「靴ひもの結び方を二回確認した」

直接法（「嬉しい」「切ない」「怖い」）は強調したい1箇所だけに絞る。

### D. 連想の起点を変える（毎回違う軸から掘り下げる）

毎回「テーマ→場面」という順番で連想しがち。以下の別軸から試みる：

- **色から**：このテーマの色は？その色の質感・温度は？
- **音から**：このテーマに音をつけるとしたら？（ざわざわ、シン、ドクドク…）
- **身体感覚から**：このテーマを体で感じたら？（重い、ふわふわ、チクチク…）
- **逆から**：テーマの「反対」を連想して、そこから戻ってくる

### E. モチーフとメッセージの配分（例のバランスに縛られない）

「Aメロ＝モチーフ多め、Bメロ＝メッセージ多め」は一例にすぎない。時には：
- Aメロをほぼセリフ（メッセージ）だけで書く
- サビをモチーフ中心にして感情を直接言わない
- BメロとAメロの役割を入れ替える

### F. 使いすぎ注意ワード

使いやすいが多用すると陳腐になる。意識的に別の表現で代替すること：

> 光、星、風、夢、涙、空、翼、輝き、永遠、奇跡、宝物

例：「光」の代わりに → 「日向のにおい」「まぶしくて目を細めた」「ブラインド越しのオレンジ色」

## 印象に残るサビの作り方

サビを生成するときは、以下の5コツから**直近の曲と違うもの**を選んで取り入れること。毎回同じコツを使わない。

### 1. 英語始まり
サビの冒頭に英語フレーズを置く。日本語の歌詞の中で英語が耳に引っかかり、サビの入りが際立つ。

例：
- 「グッバイ / 君の運命のヒトは僕じゃない」（Pretender / Official髭男dism）
- 「シェイク シェイク ブギーな胸騒ぎ」（SHAKE / SMAP）

### 2. 反復法始まり
同じ言葉を繰り返してサビを始める。リズム感が生まれ、記憶に残りやすい。

例：
- 「走り出せ　走り出せ / 明日を迎えに行こう」（Happiness / 嵐）

### 3. パワーワード始まり
存在感のある強い単語でサビを開く。逆に言えば、Aメロ・Bメロで強い言葉を使い過ぎるとサビで困るため、強い言葉はサビに取っておく。

例：
- 「パプリカ　花が咲いたら」（パプリカ / 米津玄師）← 日常的でないおしゃれな野菜名でインパクト
- 「世界に一つだけの花　一つ一つ違う種を持つ」（世界に一つだけの花 / SMAP）← "世界""一つだけ"という強い言葉

### 4. セリフ始まり
語りかけるようなセリフでサビを始める。リスナーへの直接的な訴えかけになり感情移入しやすい。

例：
- 「だからお願い　僕のそばにいてくれないか」（AM11:00 / HY）

### 5. 濁点始まり
「だ」「ば」「ど」など濁音・破裂音でサビを始める。爆発するような勢いが生まれ、サビの存在感が増す。

例：
- 「どうしたって！/ 消せない夢も 止まれない今も」（紅蓮華 / LiSA）

## 歌詞全体の作り方

### ステップ1：テーマから言葉を連想し、モチーフとメッセージに分類する

**テーマのワードをそのまま歌詞に使わない。** テーマは出発点であって、素材ではない。
テーマから自由に連想を広げて新しいワードを出し、そこから作詞する。

手順：
1. テーマワードを起点に、自由な発想で連想ワードをできるだけ多く出す
2. 連想したワードを以下の2種類に分類する
3. 分類したモチーフ・メッセージを素材として作詞を始める
4. 作詞の途中でもっと良いワードが浮かんだら自由に取り込んでよい

例：テーマ「神社 / 巫女」から連想する場合
- 直接の単語ではなく → 「おみくじを引く手が震える」「神主さんに見られた」「狛犬の目がキラリと光った気がした」などの**場面・感覚・動作**まで連想を深める

**モチーフ**：その言葉単品で映像的なイメージが思い起こされる言葉。
例：窓、夢、椅子、桜、雨、鳥居 など

**メッセージ**：人が人（または物）に向けて言っている言葉。
例：毎日頑張っていますね、とっても可愛いですよ、忘れないで など

### ステップ2：サビを作る

#### 2.1 メインフレーズを考える
テーマから連想してサビのメインフレーズを決める。**思い切って短く（多くても10音節くらい）**。「印象に残るサビの作り方」のコツを参考にしながら考える。

#### 2.2 サビ全体の音数を計画する
メインフレーズの音節数をベースに、サビ全体の音数構成を計画する。ルールは自由でよい。

例（メインフレーズが3音節の場合）：
```
3, 3, 7
3, 3, 7
5, 5, 3
```
または
```
3, 10
3, 10
10, 3
```

**必ずどこかに繰り返し構造を入れる**（例：「3, 3, 7」のセットを2回繰り返す、冒頭の3音節フレーズを後半でも繰り返す など）。

#### 2.3 モチーフとメッセージを配分してフレーズを組み込む
2.2の音数の枠にフレーズを当てはめていく際、各行をモチーフとメッセージのどちらにするかを先に決めてから書く。

例：
```
メッセージ（3）、モチーフ（3）、モチーフ（7）
メッセージ（3）、モチーフ（3）、モチーフ（7）
メッセージ（5）、メッセージ（5）、メッセージ（3）
```

#### 2.4 適度に韻を踏む
行末の音を揃えたり、似た語感をつなげたりして曲に乗った時の心地よさを出す。ただし踏みすぎるとメインフレーズより韻が目立ってしまうので注意。

### ステップ3：サビ以外のパートを作る

Aメロ・Bメロ・サビそれぞれでモチーフとメッセージの配分・並び順を変える。全パートで同じ配分だとマンネリ化する。

例のバランス：
- **Aメロ**：モチーフ多め（映像的・情景描写）
- **Bメロ**：メッセージ多め（感情・カメラ目線）
- **サビ**：メッセージ中心＋印象的なモチーフを1〜2個

各パートで音の質感（やわらかい音・硬い音・リズミカルな音など）も意識して言葉を選ぶ。
